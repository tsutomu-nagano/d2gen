# d2gen
**`d`ummy `d`ata `gen`erater**
ダミーデータを作成する

# 主な機能
- 標準記法 or 符号表の情報のJSONからデータを作成する
- 出力するレコード数を指定
- コードリストの一覧からランダムにピックアップする
- コードリストの指定がなかった場合はランダムな文字列を設定する
  - デフォルトは `A-Z` + `a-z` + `0-9` の組み合わせ
  - 使用する文字列は指定可能

## 標準記法の符号表から
- 標準記法の内容をよしなに解析してデータを作成する

## JSONデータから
- 事項の情報と符号の情報を基にデータを作成する
